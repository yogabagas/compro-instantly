# PT. Juara Satu Indonesia Website

This project is a company profile website for PT. Juara Satu Indonesia, showcasing the product "Instantly" and highlighting its B2B services.

## Project Structure

```
pt-juara-satu-indonesia-website
├── public
│   ├── index.html          # Main HTML document
│   └── favicon.ico         # Favicon for the website
├── src
│   ├── components          # Reusable components
│   │   ├── Header          # Navigation and branding
│   │   ├── Footer          # Copyright and additional links
│   │   ├── Hero            # Main message or product offering
│   │   ├── About           # Information about the company
│   │   ├── Products        # Highlights of the product "Instantly"
│   │   ├── Services        # Details of B2B services
│   │   └── Team            # Company structure and team members
│   ├── pages               # Page components
│   │   ├── Home            # Homepage layout
│   │   ├── About           # About page
│   │   ├── Products        # Products page
│   │   ├── Services        # Services page
│   │   └── Contact         # Contact page
│   ├── styles              # Stylesheets
│   │   ├── globals.css     # Global styles
│   │   └── components      # Component-specific styles
│   ├── utils               # Utility functions
│   ├── App.js              # Main application component
│   └── index.js            # Entry point of the application
├── package.json            # npm configuration file
├── jsconfig.json           # JavaScript configuration file
└── README.md               # Project documentation
```

## Features

- **Instantly Product**: A dedicated section highlighting the features and benefits of the product.
- **B2B Services**: Comprehensive details on customer relation, acquisition, service, and loyalty.
- **Company Structure**: Information about the leadership team:
  - Yoga Bagasakthi - Direktur Utama
  - Ariandi Nugraha - Direksi
  - Fitriyadi - Direksi

## Getting Started

To get started with the project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd pt-juara-satu-indonesia-website
npm install
```

Then, you can run the development server:

```bash
npm start
```

Visit `http://localhost:3000` in your browser to view the website.
